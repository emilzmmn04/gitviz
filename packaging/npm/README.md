# @emilzmmn04/gitviz

Install globally:

```bash
npm i -g @emilzmmn04/gitviz
```

The package downloads a prebuilt `gitviz` binary for your platform during `postinstall`.
Extraction is handled in Node.js and does not require a system `tar` command.

Supported platforms:

- macOS (`x64`, `arm64`)
- Linux (`x64`, `arm64`)
